# 📊 Sales Performance Dashboard – Power BI

## 📌 Project Overview
This project showcases an interactive **Sales Performance Dashboard** built in Power BI.  
It provides insights into sales trends, customer behavior, and regional performance using dynamic visuals and DAX measures.

## 🚀 Features
- **KPI Cards**: Total Sales, Total Orders, Sales per Customer
- **Bar Chart**: Total Sales by Region (with conditional formatting)
- **Donut Chart**: Top Customers by Sales
- **Line Chart**: Sales by Year (with drill-down hierarchy)
- **Treemap**: Sales by Sub-Category
- **Bar Chart**: Sales by Ship Mode
- **Matrix/Table**: Customer details with sales contribution
- **Slicers**: Year, Region, Customer Name, Ship Mode

## 🛠️ Tools & Skills Demonstrated
- Power BI Desktop
- DAX for calculated measures
- Conditional formatting & Top N filters
- Drill-down hierarchies
- Dashboard design & storytelling
- Data visualization best practices
